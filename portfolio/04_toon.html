Project Toon Shading

3D Graphics

Project Overview

This project, part of my CS250 Computer Graphics coursework at DigiPen, focuses on implementing toon shading using OpenGL. The goal was to render stylized, cartoon-like visuals and integrate the effect into an existing 3D rendering framework.

Tasks Completed

  • Computed the normal matrix for correct lighting and fog effects in D04ToonShading.cpp.
  • Updated toon.vert to transform positions and normals into view space for fog and shading.
  • Implemented three toon shading styles:
    • Custom Range
    • Uniform Range
    • Gradient Texture
  • Applied anti-aliasing to all styles.

Reflection

This project deepened my understanding of non-photorealistic rendering and the importance of normal transformations in shader programming. I also learned how to design flexible shader systems that support multiple visual styles.
